# LED Chaser

A circuit where two LEDs alternate and blink automatically using an 
astable multivibrator built from two NPN transistors and RC timing components.

## How it works

The circuit uses two NPN transistors in an astable multivibrator 
configuration. Each transistor alternately switches on and off, 
controlled by RC timing components. When one transistor is on its LED 
lights up while the other is off, then they swap — creating the 
alternating blink effect automatically without any external trigger.
